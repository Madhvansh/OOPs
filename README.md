# OOPs — Java Lab Programs

This repository contains Java programs from Object-Oriented Programming lab sessions (Lab 1 – Lab 8).

---

## Lab 1 — Introduction to Java
| File | Description |
|------|-------------|
| `HelloWorld.java` | First Java program — printing "Hello, World!" |
| `DataTypes.java` | Primitive data types and arithmetic operators |

## Lab 2 — Classes and Objects
| File | Description |
|------|-------------|
| `Student.java` | Student class with fields, methods, and grade logic |
| `BankAccount.java` | BankAccount class demonstrating encapsulation |

## Lab 3 — Constructors and Method Overloading
| File | Description |
|------|-------------|
| `Rectangle.java` | Default, parameterized, and copy constructors; overloaded `scale()` |
| `Calculator.java` | Overloaded `add()` and `multiply()` methods |

## Lab 4 — Inheritance
| File | Description |
|------|-------------|
| `AnimalHierarchy.java` | Single and multilevel inheritance (Animal → Dog → GoldenRetriever) |
| `EmployeeHierarchy.java` | Employee → Manager / Intern with method overriding |

## Lab 5 — Polymorphism
| File | Description |
|------|-------------|
| `Polymorphism.java` | Runtime polymorphism with abstract Shape hierarchy |
| `VehiclePolymorphism.java` | Compile-time (overloading) and runtime (overriding) polymorphism |

## Lab 6 — Abstract Classes and Interfaces
| File | Description |
|------|-------------|
| `AbstractPayment.java` | Abstract class with CreditCard, UPI, and Cash payment subclasses |
| `InterfaceDemo.java` | Interfaces (`Switchable`, `Connectable`) and multiple interface implementation |

## Lab 7 — Exception Handling
| File | Description |
|------|-------------|
| `BankAccountException.java` | Custom exceptions, multi-catch, and `finally` block |
| `ExceptionHandling.java` | Built-in exceptions: `ArrayIndexOutOfBounds`, `NumberFormat`, `NullPointer`, `StackOverflow` |

## Lab 8 — Collections Framework and File I/O
| File | Description |
|------|-------------|
| `CollectionsDemo.java` | `ArrayList`, `LinkedList`, `HashMap`, and `HashSet` demos |
| `FileIODemo.java` | Reading from and writing to files using `BufferedReader`/`BufferedWriter` |
